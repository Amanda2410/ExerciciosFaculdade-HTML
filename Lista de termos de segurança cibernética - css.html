Termos comuns de Segurança cibernética


  • VPN

    • -Rede Virtual Privada(Virtual Network Private). É uma tecnologia responsável pela criação de uma conexão segura e criptografada.


  • Firewall

    • -É o componente que monitora, filtra e controla o trafégo de rede, entrada e saída. Funciona como uma barreira, protegendo a rede ou dispositivo contra ameaças de segurança garantindo apenas o trafégo autorizado.


  • Antivírus

    • -É um software desenvolvido para identificar, detectar e remover os programas maliciosos, assim protegendo o sistema.


  • Malware

    • -Abreviação de "Software malicioso", é um software com intuito de danificar o sistema.


  • Phishing

    • -É um tipo de ataque de coleta de dados, onde o criminoso se passa por confiável para enganar as vítimas e obter informações pessoais e privadas.
