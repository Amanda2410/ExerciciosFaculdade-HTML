Passos necesssários para criar uma senha forte

Para que obtenha uma maior segurança é necessario que se tenha uma senha mais complexa, ou seja, mais difícil. Dessa forma, a probabilidade de invasões se tornam minímas

Observe alguns requisitos para se obter uma senha mais forte:

  1. Não use algo óbvio! Como "12345"
  2. Utilize bastante caracteres, senhas longas são fortes!
  3. Caracteres especiais, como letras maiúsculas, minúsculas e simbolos!
  4. Tente não usar sempre a mesma senha! Diversifique!
  5. Use senhas diferentes no trabalho e no pessoal
